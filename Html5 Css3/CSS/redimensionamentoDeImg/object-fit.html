Object-fit

Opa, aqui vou testar o parametro object-fit do css, todas imagens serão redimensionadas para: height - 400px e width - 500px. Lembrando, Veja o resultado abaixo

  • fill: padrão

  • contain: redimendionado mas mantem as proporções

  • cover: a imagem é cortada nas proporções dadas

  • none: a imagem não é redimensionada

  • scale-down: a imagem é redimensionada para a menor escala do none ou cover
