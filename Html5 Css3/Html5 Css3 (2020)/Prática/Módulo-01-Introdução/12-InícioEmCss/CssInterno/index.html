Nesta página estou usando estilos css internos

Para mudar o estilo dessa página eu usei css interno, onde dentro da tag head eu criei outra tag chamada style e modifiquei o estilo da pagina de uma forma bem mais simples que o css inline

Vantagens e desvantagens desse método

A grande vantagem desse método em relação ao inline é a organização do código e a aplicação de um código só para varios elementos, porém tem a desvantagens de que o estilo pode ficar maior até mesmo que o código do body, para isso existem css externos que veremos no próximo arquivo
