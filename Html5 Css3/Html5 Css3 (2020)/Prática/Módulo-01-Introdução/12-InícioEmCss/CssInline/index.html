Nesta página estou usando estilos css inline

Para mudar o estilo dessa página eu usei o css inline, onde dentro da tag html eu coloco um parâmetro style="estilo" e uso as css para alterar.

Evite usar esse método

Não se recomenda usar este método devido a vários motivos, porém, os principais são a repetição de código, porque não é possivel aplicar varias tags a uma mesma linha, e a poluição do cógido, pela mistura entre html e css além de dificultar o entendimento do código.
