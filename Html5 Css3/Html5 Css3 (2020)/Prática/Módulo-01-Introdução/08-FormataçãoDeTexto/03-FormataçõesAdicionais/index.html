Algumas formatações extras

Texto marcado

Podemos criar um texto marcado usando a tag mark

Texto grande e pequeno

Podemos criar um texto grande usando a tag big ( obsoleta )

Para criar um texto pequeno usamos a tag small

Texto deletado e texto inserido

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado, e para isso usamos a tag del

Podemos marcar um texto como inserido para dar uma ênfase de que foi adicionado depois. E para fazer isso usamos a tag ins

Texto sobrescrito e texto subscrito

Podemos criar um texto sobrescrito para escrver coisas como x2-4=0, para isso usamos a tag sup

E um subscrito para coisas como H2O, para isso usamos a tag sub
