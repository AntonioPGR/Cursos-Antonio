Trabalando com modelo de caixas

No html5 temos dois tipos de modelos de caixa, os box-level e os inline-level. A tag <h1> é um exemplo de box-level, pois ocupa a linha toda, já os links, são exeplos de tags inline-level.

Atributos de uma caixa

Podemos dividir uma caixa em 5 partes, são elas:

  • Conteudo: parte central onde se insere o conteudo da caixa
  • Padding: parte que circula o conteudo e regulo o espaço entre ele e a borda
  • Border: parte a qual circula o padding e define os limites da caixa
  • OutLine: parte que circula o Border. Ela funciona como uma borda externa que ocupa o espaço do margim, ela não possui um espaço próprio
  • Margim: parte que circula a borda e define uma margem diante a outra caixa que ficará próxima.

Tipos de caixasBox Level

  • Box-level: é uma caixa onde ela, por padrão, ocupa totalmente a linha e fica 'isolada', como o <h1> acima.
  • Inline-level: já esse outro tipo compartilha a linha com outros elementos, como a tag <a> destacada acima
