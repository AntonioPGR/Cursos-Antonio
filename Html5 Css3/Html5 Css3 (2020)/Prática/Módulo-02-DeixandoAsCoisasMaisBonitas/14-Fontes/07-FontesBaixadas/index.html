Utilizando fontes baixadas

Para utilizar fontes baixadas fazemos o seguinte:

  1. Baixamos a fonte para nossa maquina, existem sites que fazem isso, um exemplo é o dafont.com
  2. Colocar o arquivo dentro da pasta onde está seu site
  3. Crie uma regra no seu css que será o seguinte: 
    @font-face {
    font-family: 'nome da fonte';
    src: url('Local do arquivo') format('formato do arquivo'), url('Local do arquivo') format('formato do arquivo');
}
  4. Você pode adicionar mais de um formato de fonte para ter mais compatibilidade com outros navegadores.
  5. Sobre o Format(), existem 5 tipos:
    • opentype (geralmente arquivos .otf)
    • truetype (geralmente arquivos .ttf)
    • embedded-opentype
    • truetype-aat (Apple Advanced Typography)
    • avg
      •
  6. Pronto! agora você tem a fonte que tu queria em seu site!
