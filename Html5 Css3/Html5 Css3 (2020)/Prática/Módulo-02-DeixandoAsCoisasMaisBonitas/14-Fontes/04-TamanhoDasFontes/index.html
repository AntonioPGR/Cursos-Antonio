Tamanho de fontes

O tamanho de uma fonte pode ser alterado de maneira simples através do comando: font-family: tamanho da fonte;. Porem, uma das coisas que é muito controversa é que medida usar no tamanho de uma fonte. Para te falar, vou primeiro te mostrar as medidas possiveis. As medidas são dividias em dois tipos:

Absolutas

  • cm
  • mm
  • in ( polegada )
  • px
  • pt
  • pc

Relativas

  • em
  • ex
  • rem
  • vw
  • vh
  • %

Mas afinal, qual usar?

Bem, sobre isso, muitos programadores usam medidas variadas, porém, o w3c, orgão que cuida do html e do css, recomenda que se usa principalmente o em, e tambem se pode usar o px, mas dando preferencia ao em. Agora, em base de que o em trabalha? bem, ele é uma medida relativa a fonte original, e, na maioria de seus navegadores a fonte padrão é 16px, então, 1em = 16px, assim como 2em=32 e tbm 1.5em=24px.
