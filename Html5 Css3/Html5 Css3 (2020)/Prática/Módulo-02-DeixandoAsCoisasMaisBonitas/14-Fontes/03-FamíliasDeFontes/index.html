Famílias de fontes

Para declarar que um texto pertence a certa família de fontes em css é muito simples, basta usar o parâmetro:
font-family: fonte1, fonte2, tipo da fonte

Sobre os valores acima eles funcionam da seguinte maneira: o navegador ira primeiro tentar a fonte um, se não conseguir, irá para fonte dois, e se não conseguir, irá para uma terceira fonte, e fará isso dai em diante com todas as fontes que você por. O ultimo paramêtro geralmente se põe uma 'classe' de fonte, pois se o navegador não conseguir carregar nenhuma fonte, ele pegará uma fonte padrão daquele tipo. exemplo: serif, sans-serif, monospace,
