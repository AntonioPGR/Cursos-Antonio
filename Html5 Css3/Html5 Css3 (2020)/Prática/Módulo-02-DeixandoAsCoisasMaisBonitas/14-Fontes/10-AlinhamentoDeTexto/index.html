Alinhamento de Texto

O que é?

Bem, quando escrevemos um texto queremos que ele, alem de informativo, fique bonito. Claro, para ficar muito bom se deve usar varias propriedades, porém, o alinhamento é uma deles. Um texto bem alinhado deixa tudo muito melhor e mais claro, e para fazer isso em css é muito facil, basta usar o parâmetro text-align: valor;. Para o valor que ira ser passado, temos varios tipos, veja abaixo:
