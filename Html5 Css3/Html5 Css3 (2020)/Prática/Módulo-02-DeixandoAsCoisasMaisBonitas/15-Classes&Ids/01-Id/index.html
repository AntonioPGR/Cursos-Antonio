Utilizando ids com css

Quando aplicamos um id a uma tag html, usando: id="x", estamos dizendo que aquela tag se identifica através x, então, podemos usar um estilo css baseado em x: h1#x {}, onde a # significa id. Veremos mais para frente que se tem tbm outros identificadores como o '.' para a classe, mas deixa isso para depois. abaixo temos textos inuteis só para você poder ver o uso do id nesse paragrafo superior e no inferior não
